# jsLife

Реализация игры «Жизнь» Конвея на чистом JavaScript с 5 цветами клеток.

## Запуск

Откройте `index.html` в браузере или поднимите статический сервер:

```bash
python3 -m http.server 8000
```

После этого перейдите на `http://localhost:8000`.

## Правила в этой версии

- Есть 5 цветов живых клеток.
- Каждая живая клетка учитывает **только соседей своего цвета**.
- Живая клетка выживает, если у неё 2 или 3 соседа своего цвета.
- Пустая клетка рождается цветом, у которого ровно 3 соседа вокруг.

## Управление

- **Старт / Пауза** — запускает и останавливает симуляцию.
- **Шаг** — выполняет одно поколение (когда пауза).
- **Очистить** — очищает поле.
- **Случайно** — заполняет поле случайными цветными клетками.
- **Скорость** — настраивает частоту обновления.
- **Палитра цветов** — выбирает цвет, которым рисуются клетки по клику.
- Клик по клетке ставит выбранный цвет, повторный клик этим же цветом очищает клетку.
